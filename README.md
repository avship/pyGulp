# pyGulp

Столкнулся с проблемой. Есть старенький нетбук (да, тут не описка, именно нетбук). Очень хочется его использовать в дороге, но есть беда: под windows 7 32 бит ставится только старенькая node js. В связи с этим оказывается невозможно использовать gulp. 

Данный проект призван решить эту проблему. В текущей версии программа имитирует работу модуля gulp-file-include. Шаблоны проекта, которые будут подключаться к вашим файлам, лежат в папке templates. Вообще всю разработку ведём в папке src. 

Пока что модуль работает только с html файлами, но в следующих обновлениях будет реализована работа с js и less
